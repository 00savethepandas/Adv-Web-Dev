A fade-transition Backbone carousel

Define a container, and place carousel items inside, each with a class of carousel-item. Clicks on anything with class carousel-prev and carousel-next will advance the carousel. Pass a selector for the top-level container into the constructor as el.

There is minimal style here. It's only important to hide all of the carousel-items and allow the Backbone view to manage their visibility.

The carousel can be set to auto-advance, by calling startRotating(). Or the user can be in control using the toggle button.
